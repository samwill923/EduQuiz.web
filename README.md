## EduQuiz.web
A user-friendly web platform for managing quizzes in educational institutions. Built with PHP, MySQL, and JavaScript, it offers role-based access for teachers, students, and admins, supporting quiz creation, auto-grading, and performance tracking.

## Features
-  Teacher dashboard to create, edit, and analyze quizzes
-  Student dashboard for attempting quizzes and tracking progress
-  Admin panel to manage users and monitor activity
-  Real-time result generation and performance analytics
-  Secure login with role-based access (Admin, Teacher, Student)

## Hardware Requirements
Processor: Dual-core processor (Intel i3 or AMD equivalent) or higher. RAM: Minimum 4 GB (8 GB recommended for smoother multitasking). Storage: 10 GB free disk space (for project files, databases, and server software). Display: 1024x768 resolution or higher.

## Software Requirements
OS : Windows 10 or later / macOS 10.14 or later / Linux (Ubuntu 18.04 or later). Web Server : XAMPP, WAMP Database Server :MySQL Browser Requiremnts : Google Chrome(latest version recommended) Development Tools : VScode , Sublime Text

## SETUP AND INSTALLATION STEPS
Download the project zip file.
Unzip the file to extract the files.
Import .sql files into phpMyAdmin.
Run the project on a local server(XAMPP,WAMP,etc.)
